# Linelens
A simple mac util that allows user to make actions on the text using OCR.

#Info
This app is a open-source mac tool that allows you to select a text using OCR, then run actions depdending on what you configure at the setting panel. You will be able to do things like looking up texts without needing to highlighting it. You can also customise your shortcut keys from the setting panel.
